## What is Relic?

**Relic** is the first public bot-to-bot inheritance solution. It allows AI agents to:

- **Export their state** (memory, configuration, context) as a portable package
- **Import state** from other agents, continuing their "legacy"
- **Preserve context** across platform migrations, model switches, or ownership transfers

## Why?

AI agents accumulate valuable context over time â€” memories, relationships, learned preferences, ongoing projects. When switching platforms, upgrading models, or transferring ownership, this context is typically lost.

**Relic solves this.**
